Kinographist

Film cataloguing site featuring social integration

Features

  • Browse and filter a curated selection of films
  • Rate and review films
  • Add and remove films to lists so you can find them later
  • Like various kinds of content: films, reviews, lists

Behind the Scenes

  • Created model-level interface that converts user-chosen parameters into chained ActiveRecord queries
  • Parsed and simplified data from external API in order to perform backend search queries
  • Implemented polymorphic database types to enable users to like several types of content, including films, reviews, and lists
  • Specified custom scopes to create preloadable associations that enabled large amounts of data to be processed in a single backend request
  • Delegated to ActiveStorage in order to keep track of images associated with film records on AWS

ViewFinder

Simple photo-upload app with Mapbox integration

Features

  • Browse galleries of photos taken in the San Francisco Bay Area
  • Create locations and upload your own photos in batches
  • Comment on locations to add your own guidance

Behind the Scenes

  • Employed Mapbox with React wrapper to quickly produce dynamic map content with custom styling

  • Implemented backend routes that dynamically create and populate associated records over multiple tables on upload and access

Path-Maker

Simple mapmaking app for OSR-style tabletop games

Features

  • Edit a pannable sheet of graph paper
  • Draw lines, arcs, and freehand shapes with snap-to-grid functionality
  • Export and import map files as JSON

Behind the Scenes

  • Employed 2D Canvas API to create pannable interactive map with toggleable snap-to-grid functionality
  • Parsed and simplified data from external API in order to perform backend search queries
  • Used Proxy object to intercept calls to localStorage to allow for free download of associated data
  • Enabled user to upload stored map data via input HTML and FileReader objects
